Projects

Ride Share

My final Ruby project, where I applied my knowledge of oject oriented programming to create a ride share system from data given in CSV files. By implementing a rider, driver, and trips class, each with its own attributes, class methods, and instance methods, I was able to build a system to manage the given ride sharing information.

Scrabble

My second pair programming project, where my partner and I practiced using TDD to create a collection of Ruby classes that would provide the foundation for a scrabble game. We chose to implement a tile, tile bag, and player class as well as attempted to use our recent knowledge of enumerables and ternary operators to DRY up our code without compromising readability.

MeowSpace

My first tackle at building a static site from a provided wireframe using only HTML and CSS. The website is optimized for a 13" screen and a Google Chrome browser. Future improvements include making the webpage more responsive to the given screen size and browser.
